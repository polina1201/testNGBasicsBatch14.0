TestNG
TestNG is a tool used to perform unit and integration testing.
Execution is done alphabetically, rather then top to bottom.

Data Driven Testing
    When one Test cases executes against multiple set of data

Annotation
    @Test - it is a chief annotation of testing and every TC starts with it
    By default, it follows alphabetical order for execution.

@BeforeMethod
@AfterMethod

@BeforeClass

=if else statements use in java/selenium

Verification vs Validation
Verification - Checking the existence
Validation - Checking the correctness of the field after existence

Assertions in TestNG
    Hard Assertion
    Soft Assertion

Hard Assertion
Whenever assertion fails in hard assertion - it will stop the execution
